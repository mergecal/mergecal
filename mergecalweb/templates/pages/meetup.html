Smart Meetup Calendar Integration

Keep your calendar in sync with your favorite Meetup groups - even when their iCal feeds don't work.

The Problem: Meetup's iCal links (ending in /events/ical) only work as one-time downloads. They won't sync with your calendar app.
Our Solution: We use Meetup's REST API to fetch live event data and convert it to a proper, always-syncing calendar feed.

How It Works

Example Transformation

When you add this Meetup URL to MergeCal:

https://www.meetup.com/baltimore-code-and-coffee/events/ical

We:

  1. Extract the group name (baltimore-code-and-coffee)
  2. Fetch real-time data via REST API
  3. Convert it to a proper calendar feed
  4. Keep it synchronized automatically
You get a reliable calendar feed that actually updates - perfect for following multiple tech meetup groups.
